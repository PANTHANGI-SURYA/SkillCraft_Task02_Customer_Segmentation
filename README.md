# SkillCraft_Task02_Customer_Segmentation
This project implements a K-Means Clustering algorithm to segment customers of a retail store based on their purchasing behavior. Using the Mall Customer Dataset, the model analyzes key features such as Annual Income and Spending Score to group customers into meaningful clusters.
